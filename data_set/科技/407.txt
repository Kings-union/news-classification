Ta是十足生产力工具 15寸惠普Spectre x360评测
　　本文来自太平洋电脑网
　　大家对于惠普Spectre x360这个产品也应该不会很陌生，它是惠普高端产品线的典型代表，是享受品质生活的极佳选择。这次发布的15寸惠普Spectre x360，迎来了重大升级，不仅继承了这个系列的轻奢优雅的外观，而且在配置方面也有了重大升级。最大的亮点是Intel Core i7-8705G处理器，这款处理器集成了AMD的核心显卡，性能比某些独立显卡的还要强一些，让惠普Spectre x360拥有了优秀的图形处理能力。15寸的宽广机身可以让用户尽情挥洒脑海中的创意，并且进一步拓宽了视野。它虽然是隶属于轻薄本产品线，但确实一个十足的生产力工具。
　　外观依旧充满贵族气质：　　
　　我们看一下这台15寸的惠普Spectre x360会给我们哪些不一样的惊喜。为了给用户打造出细腻上乘的手感，惠普Spectre x360的机身采用的都是全铝材质打造，并且采用的是CNC钻石切割工艺，足见惠普在选料做工上的大手笔。整个机身重量为2.1kg，机身厚度为19.45mm。惠普Spectre x360在15寸的机身下，还能保持住这样的厚度和重量，也是非常难能可贵的。在配色方面，这款惠普Spectre x360依旧选择的是摩卡黑和青奢金的配色，为整台机器增添了些许贵族气质。
　　而15寸的惠普Spectre x360在屏幕设计上也力求突破。它的屏幕是15.6寸的UHD全高清触控屏，分辨率为3840×2160（4K），亮度340尼特，NTSC色域高达72%，可以使图像色彩还原的更加逼真，细节展示更加细腻。虽然惠普Spectre x360的机身为15寸，但是它的屏幕依旧是窄边框设计，边框宽度仅为7.04mm，这样的设计在其他的15寸笔记本中算是很少见的，能极大限度的拓宽用户的视野。此外屏幕边角采用还是小弧度设计，显得更加富有力量感。
　　让我们再来了解一下惠普Spectre x360的键盘体验。它的键盘拥有1.5mm的键盘，输入起来极为舒适。15寸的广阔机身，它当然配备小键盘，输入更加自如。另外它的键盘配色为黑底白字，背光颜色也是白色，和机身的颜色极为搭配，营造出非常舒适的视觉感受。
　　我们再来看下惠普Spectre x360的拓展性如何，很多轻薄本为了拥有更便携的身形，所以在拓展性上就做了很大的牺牲，而新款的惠普Spectre x360完全不用在意这个问题，它的拓展接口相当丰富。它拥有1个USB3.0 接口、2个Type-C接口、1个HDMI接口、1个SD卡槽和1个3.5mm耳机接口，这样的设计可以满足用户外接存储设备、进行屏幕拓展等常用需求。
　　最后我们再来看一下15寸的惠普Spectre x360相比于其他同类机型比较有特色的部分。这台机器最具特色就是它拥有高精度的联动式铰链，让它的机身可以进行360度旋转，屏幕支持10点触控，同时它随机配备的触控笔响应速度快、延时低，非常适合专业的设计师使用。
　　性能有了质的飞跃：
　　CPU-Z测试：
　　现在很多轻薄本处理器，大家最熟悉的可能是Intel Core i5-8250U、Intel Core i7-8550U这些型号。而这台惠普Spectre x360的处理器型号却比较特别，是Intel Core i7-8705G处理器。这款处理器就是Intel和老对手AMD合体的结果。这款处理器最大的特点就是集成了Radeon RX Vega M核显，采用EMIB技术，将CPU、核显与HBM2高速串联，拥有体积小、能耗低和高效率表现。集显我们待会再讲，这里我们先看一下Intel Core i7-8705G的基本参数。Intel Core i7-8705G拥有4核心8线程，基础频率3.1GHz，三级缓存为8MB，热设计功耗为45W，采用的是Kaby Lake-G架构和14nm的制作工艺。从上述参数我们就可以体会到，这是一款高性能的处理器。
　　同时我们用了CPU-Z这款软件检测了惠普Spectre x360的内存情况。很多轻薄本为了节约能耗，一般都用的是DDR3内存条，但是由于频率比较低，非常影响数据的传输效率。而惠普Spectre x360配备的是16GB的DDR4内存，频率为1130MHz。相比于DDR3内存，它的性能提升了不少。
　　CINEBENCH R15测试：
　　光看处理器参数可能比较抽象，我们用了CINEBENCH R15这款软件测试了Intel Core i7-8705G的性能，这款处理器的多核处理能力得分为784cb，单核处理性能得分为161cb。很多轻薄本常用的Intel Core i7-8850U处理器在这个软件里跑分大概为520cb，这样看来Intel i7-8705G已经超出了它们不少，可以为惠普Spectre x360注入更澎湃的动力，让它更能胜任一些高负载的生产力任务。
　　GPU-Z测试：
　　我们再来重点介绍一下Intel Core i7-8705G处理器所搭载的AMD集显。从GPU-Z检测出的数据来看，这款集显的名称为Radeon RX Vega M GL Graphics，它的显存容量为4GB，显存类型为HBM2，显存位宽为1280bit，总线接口PCIe，基础频率为931MHz，加速频率为1101MHz。总体来说，这款显卡比Intel处理器集成的其他显卡要强许多，甚至比一些早期的独显性能还要强。
　　3DMark基准测试：
　　说到底，Radeon RX Vega M GL Graphics的性能究竟有多强呢？我们用3DMark测试它的图形处理能力，这次我们选择了三个模式，分别为Sky Diver、Fire Strike、Fire Strike Ultra。
　　Sky Diver模式主要是测试中端电脑，这个模式也主要用于测试一些轻薄本，而在这个模式下，惠普Spectre x360的得分已经高达了18041。相比之下，同类型搭载Intel Core i7-8550U的电脑，在这个模式下的得分为5000分左右，可见Radeon RX Vega M GL Graphics集成显卡将电脑的图形处理能力提升不少。
　　而Fire Strike模式则主要测试的高性能的游戏电脑，而这台惠普Spectre x360的得分为6273分。这个分数在同类型的产品也算是不错的了，可以让它省一些中端的网络游戏。
　　Fire Strike Ultra主要是测试的电脑的4K性能，我们也测试一些供大家参考，惠普Spectre x360的得分为1620分，表现依然不错。
　　PCMark基准测试：
　　说到底，这台惠普Spectre x360的性能到底强在哪里？我们还需要做一个全面的评测，PCMark 8可以测试电脑的存储、计算、游戏、图形处理等诸多方面的性能，最后给出一个评分，而这台惠普Spectre x360的得分为4012分，证明它完全有能力处理一些日常任务。
　　续航能力测试：
　　作为一台生产力工具，惠普Spectre x360肯定免不了经常运行一些高负载的任务，那么它的续航能力是否就逊色了呢？答案是否定的。这台惠普Spectre x360的电容量为82790mWh，比轻薄本的电量多了30%左右，拥有了大尺寸机身就拥有了更大容量的电池。我们用PCMark 8对惠普Spectre x360进行了续航能力专线测试。为了接近真实使用情况，我们将亮度调到50%，这台电脑的成绩3小时21分钟，与同类型产品的成绩相近。
　　硬盘读写速度：
　　硬盘读写速度也非常影响电脑的性能，这台惠普Spectre x360搭载了容量为1TB的NVMe版的固态硬盘，既保障了传输的速度，又兼顾了容量。NVMe是一种新的硬盘传输标准，它拥有更低的延时、更高的传输速度、更低的功耗。在AS SSD Benchmark的测试中，惠普Spectre x360的连续读取速度为2195MB/s，连续写入速度为344MB/s，综合得分为1819分。这样表现已经算是相当不错，可以为惠普Spectre x360提供强劲的性能。
　　使用体验极为舒适：
　　坚实的安全屏障：
　　这台惠普Spectre x360相比于之前的产品，拥有非常的高安全性。它除了可以设置大家非常熟知的Windows登陆密码之外，它支持面部识别和指纹识别，这将惠普Spectre x360提升了一大步，这样的设计除了机主本人可以登陆之外，其他人很难破解，为保护隐私安全和数据安全提供一道非常坚固的屏障。而且相对于传统登陆密码，面部和指纹的识别率高、反应速度也更快。
　　此外它还支持图片密码、随机码等其他登陆方式，总有一种登陆方式符合你的使用习惯，并且安全系数也是非常高。
　　处理工作游刃有余：
　　15寸的惠普Spectre x360处理日常工作的表现怎么样呢？我们用了大家熟悉的压缩文件这个功能做了下测试，操作虽然简单，但是要是快速地解压大文件，也需要配置比较好的电脑。在解压一个将近4GB的文件，惠普Spectre x360耗时为2分钟左右；在压缩这个文件的时候，耗时为5分钟左右。这比其他搭载低压处理器的轻薄本快了不少。
　　十足的生产力工具：
　　15寸的惠普Spectre x360拥有如此大气的机身和如此强劲的配置，一个重要的目的，就是为了可以胜任更多高负载的生产力工作。我们在测试过程中，用到了画图和OneNote这两个软件，结果发现惠普Spectre x360的响应速度非常快，手写笔用起来也很顺手。资深的设计师要是选择它来工作，绝对是一件非常趁手的兵器。

　　PConline评测室总结：
　　总体来说，这台惠普Spectre x360给我们带来的惊喜还是很多的。15寸的机身不只是身材上简简单单的扩大，让设计师尽情的挥洒自己的创意；把它当做娱乐工具时，还可以拥有更广阔的视野。在外观设计上，依然保持了窄边款屏幕、轻巧身形等之前产品的优点。惠普Spectre x360搭载的Intel Core i7-8705G处理器，让整台机器的性能有了质的提升，让它可以负载更多的高负荷任务。惠普这次产品的升级，让传统的轻薄本蜕变成了强劲的生产力工具。
