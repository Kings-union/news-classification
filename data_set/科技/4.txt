硬件大升级后 全能笔记本Surface Book2已经没有对手
　　本文来自爱活网
　　在两年前的第一代Surface Book上，微软让我们看到了个人笔记本的理想形态。要说当时留有什么遗憾，大概只有性能尚有提升空间和USB Type-C接口缺席。两年时间只解决两个问题，可能你会觉得微软动作太慢。然而，接近完美已经很难，要在这个基础上再前进，难上加难。
　　所以，为Surface Book 2等待的这两年，究竟值不值得？
　　完美的部分依旧完美
　　到达爱活的Surface Book 2是规格最高的15英寸款，可谓Surface Book 2的完全体。和上一代一样，Surface Book 2仍然有着在第一眼就吸引住直男的外表——尽管仍然没有超窄边框屏幕，但Surface Book 2的镁质一体化机身给人的品质感，依旧经得住时光的考验。
　　正如上面所说，Surface Book 2的A面、C面、D面保持了100%全金属表面的设计，视觉观感上无可挑剔，摸上去皮肤与之接触的沁凉和特有的金属质感也绝对不会让人觉得其档次低。想象一下眼看夏天来了，穿着舒服透爽的裤子，Surface Book 2放在大腿上，才有2kg不到的重量，甭提多惬意。基本上Surface Book 2这样的全金属设计，唯一存在的担心是在冬天可能对手腕不够友好。
　　而在其展示过独门绝技之后，妹子也会对它投来赞许的目光——Surface Book 2继承了上一代的优秀形态转换能力。从底座上脱离，它是一块平板电脑；插入底座，它又变身为更接近传统笔记本的形态。Surface Book初代登场时的独特思路，指引了一部分做变形本的小伙伴，不过它们都只抄其形未得其精髓。
　　Surface Book的形态转换能力来自于其独特的动态支点铰链结构，Surface Book 2同为如此，这一设计仍然能为重量分配更偏向屏幕部分的Surface Book 2提供足够的支撑，不论在桌面使用还是在大腿上使用，笔记本整体的结构稳定性依然是上佳的，只要不动手戳屏幕，根本不会察觉到它会产生任何抖动。
　　Surface Book 2的分离插入操作也和初代目完全相同，插入很好理解，平板带有三个接口的那端对准底座上的三个插头，直接放入即可；分离操作则不那么直觉一些，需要先按下键盘右上角的分离按键，等绿灯亮起，铰链中发出咯哒的声音后即可安全把平板部分取出。不过，底座和平板的固定并不是卡死的，如果稍用点力气仍然能把平板从底座上拽出来，这里面可能存在损坏设备的危险。
　　因为协调机身重量的需要也在，Surface Book 2的平板模式不能长时间承受密集计算负载，屏幕部分虽然是机身核心但没有独立GPU。其实主要还是用起来有些吓人，15寸屏幕大得如同餐盘，没有双手握紧的话心里超级慌张害怕拿不稳打翻。尽管如此，Surface Book 2平板模式能完美发挥那3：2比例屏幕的2K分辨率的精细和视野，来浏览网页，观看视频和做PPT演示都是种享受。反倒是桌面模式下UI DPI缩放的问题在一些应用上仍会碰到（如Photoshop CC的保存文件界面）。
　　和上一代一样，Surface Book 2无法像传统笔记本那样完全闭拢，B面和C面之间会存在一个楔形空缺。可能很多喜欢齐齐整整的人并不是很喜欢这种设计，但它对Surface Book 2来说又是必要的，不然无法给手感完美的键盘留出空间。
　　与Surface Book 2的人机交互可能是我所接触过的笔记本里面最能让我感到自然舒适的。先不说第一眼看过去完全统一的金属色，和底座融为一体的美感就很引诱人上手，按键软硬适中，在其上运指如飞没有机械键盘那种锻炼手指肌肉承受能力的感觉。它先会接住你的敲击，接近键程末端时给手指充分的缓冲，让你感觉自己不是在敲一块钢板，但并非那种毫无反馈的肉感——不是硬梆梆的肌肉，而是刚柔并济的软妹手，敲一句话的功夫，直男的心就已经被俘获了一半。
　　Surface Book 2的触摸板也是如此，虽然说和Macbook比起来总是萝卜青菜，但我个人更喜欢Surface Book 2触摸板的玻璃覆盖层带来的顺滑移动，在其上使用触控手势轻松惬意，速度精度都在笔记本模式下哪怕没有鼠标也不会感觉到自己会被操作束缚手脚。
　　最后，有必要说一下Surface Book 2整机的便携性，因为新增了15寸款，还有下文要提到的硬件性能升级，Surface Book 2的电源适配器功率也增大到了102W（笔记本95W，外加一个5V/1.5A的USB供电口），这也意味着重量增加——虽然它大小似乎没有什么变化，体重是实实在在地从250g长到了355g，加上本体1.9kg的重量，全重携带15寸款Surface Book 2为2.25kg，其实对15寸本来说是个很优秀的数字。不谈别的，就用那些主流15寸游戏本来比较，这体重已经是少掉一个板砖电源的重量了。
　　性能不再留存遗憾
　　Surface Book初代之所以会给人留下遗憾，理由和Surface Studio差不多，就是硬件配置上总让人觉得差口气，虽然它确实还是笔记本的身份，但独立显卡后面的那个M在傲然立于笔记本巅峰的王者身上总让人感觉不自在。
　　两年时间过去，PC硬件也出现了大跨步的升级。Surface Book 2则刚好赶上了最合适的时机，Intel第八代Core发布，NVIDIA Pascal架构显卡也日臻纯熟，这两样装备的加身让Surface Book 2的性能表现相比初代目是脱胎换骨式的。前文提到爱活站内的Surface Book 2为15寸顶配型号，它搭载Intel Core i7-8650U处理器，16GB RAM，1TB NVMe SSD，以及NVIDIA GeForce GTX 1060 6GB GDDR5独立显卡。没有M，没有MAX-Q，只有毫不妥协的性能诉求。
　　多出的两个CPU核心，以及更高的运算频率都给Surface Book 2带来了不小的纸面实力加强。在多次PCMark 10的基准测试中，Surface Book 2的得分都在4100分左右，是八代低压CPU+NVMe SSD系统的普遍水准；而3DMark Time Spy的反复测试以及Fire Strike参考测试里，Surface Book 2成绩分别在3600/9000左右徘徊，可见其游戏性能其实和主流桌面游戏系统没什么差距了。
　　另外我还顺手用CrystalDiskMark 6.0.0测试了一下Surface Book 2里那块三星PM961 NVMe SSD，2854.6MB/s的顺序读取速度和高达186872.8IOPS的4K随机性能（Q8/T8下）都让人瞠目结舌。
　　在实际环境中，CPU和GPU升级所获得的工作效率体现也更加显著。笔者在Surface Book 2上使用Adobe Lightroom批量输出19张富士X-T2相机拍摄的RAW照片，导出速度能和我那台祖传的Core i7-4790K/GTX 970的台式机齐头并进，1分20秒搞定；随后我又用Adobe Premiere Pro以H.264格式输出了一下以前做的时长为7分39秒的720p自制游戏攻略视频，它也只用了两分钟不到的时间就完成任务，在顶配台式机面前也并不逊色。
　　有了一块不打折扣的显卡，当然要用游戏来考验实力，我在它上面小试了一下最近玩得比较多、或较新潮的几个游戏，分别是《绝地求生》、《刺客信条：起源》、《最终幻想15》以及《文明6》，四款不同风格不同类别，计算力消耗侧重也不太相同的游戏。所有游戏画质选项全部都设为最高。
　　首先是国内热度仍然未减的《绝地求生》，因为没有预先提供的测试流程，这里我使用了Fraps帧数记录来测定跳伞落地后战斗过程的游戏帧速，可以看到在多数时间，Surface Book 2都能保证游戏在2K分辨率下的流畅运行（游戏最高只能支持到2560×1600），平均帧率可达81fps，干活干累了，用Surface Book 2吃个鸡也没有问题。
　　接下来我试着玩了一会刚刚上线不久的《最终幻想15》PC版，从画质角度而言没有比它更能考验一台游戏PC，同样这里记录一段主角到达锤头鲨MOTEL之后的帧率，一开始我使用Surface Book 2的原生分辨率3240×2160进行游戏，但游戏实时演算过场动画的运行情况比幻灯片还要卡，看来GTX1060对付不了火力全开的FF15，把分辨率下放到2560×1600，游戏基本上可以没有什么障碍运行了，不过平均帧率大概只有23~24fps左右。
　　然后是埃及旅游游戏《刺客信条：起源》，游戏识别了Surface Book 2的原生分辨率，此处我记录的是开场的两场战斗以及之后一小段场景探索过程的帧率，Surface Book 2的平均帧率表现在20fps左右，也是勉强可以进行游戏的水平，但想流畅需要考虑降低分辨率到2K或调低画质。
　　最后是老年人健脑娱乐游戏《文明6》，该作其实相比GPU更注重CPU一些，游戏后期的每回合AI行动的计算都是CPU完成的。《文明6》也提供了跑分选项，此处我使用程序提供的标准化流程，更具参考意义。《文明6》评分分为图像和AI两项，Surface Book 2的打分则在25fps帧率和15s/每回合附近。
　　可以这么说，主流竞技对战游戏对Surface Book 2来说是小菜一碟，而3A大作只要你肯屈尊让它跑在1080p分辨率上，仍然可以高画质设置下流畅享受。
　　说完内部硬件，还要提一句Surface Book 2的升级也体现在接口上，原来的MiniDP变成了USB Type-C——虽然微软没有在纸面参数表上写明，但该接口支持USB Power Delivery特性，既可以通过该接口给连在它上面的设备充电，也可以从这个接口获取电力。在应急情况下你是可以使用功率足够的充电宝为它充电的，但不要指望能够驱动这台性能巨兽全速运转。
　　此外，其他I/O接口配置和初代Surface Book没有太多变化：两个USB 3.1 Gen 1接口，一个SD卡读卡器，以及Surface Connect磁性充电口。如果需要更丰富的扩展性能，那就得去买微软准备的USB-C转HDMI/VGA转接器，或者是Surface扩展坞了。
　　通向下一个完美的那些路标
　　当然，强如Surface Book 2也无法做到尽善尽美，它身上还是有两三处可能会让人感到“要是能XXX就更好了”的地方。之前那个插电源玩游戏掉电的问题讲起来好像很糟糕，在使用过程中反而影响是最小的，因为该现象在电量达到90%以下之后基本上就会停止，也就是说Surface Book 2并不会挂着交流电输入却把电池榨干。
　　其实Surface Book 2有两个比这问题更容易察觉的美中不足，比较小的那个是发热和噪音。如果让其玩大型游戏或是视频编辑这样的繁重任务，Surface Book 2的发热会变得明显，尤其是容纳了独立显卡的底座，使用时尽量避免把底座靠近转轴的那一半贴着裸露的大腿，不然接触的皮肤区域容易感到闷热。平板部分反倒散热比较均匀。而且在这种情况下Surface Book 2散热系统也会全速运转起来，若是大白天环境噪音比较大可以无视，但在安静的夜晚把耳机摘下来，风扇努力的声音还是清晰可辨的。
　　另一个相对易察觉的美中不足之处是Surface Book 2的续航，根据我每天工作时段里照片处理工作量的多寡，Surface Book 2的续航时间会在8个半小时到10个小时不到之间波动——典型情况就是上午10点半开机，下午6点一刻左右提醒需要充电，就算让它全天只做网页浏览和文字编辑的轻活，它也很少能撑过10小时。如果是朝九晚五通勤一族的话，其实这没有什么影响，只要不加长班，不带充电器出门上班也可以。实话说，并不是批它续航能力差，只是和微软官方提供的13小时参考比一下落差有点大。
　　另外我也特意试了一下Surface Book 2的视频播放续航能力，在Surface Book 2两块电池余量都在60%的下午5点50分左右时开始播放1080p格式的日剧，最后看到11点20分笔记本提示电量严重不足，需要立即充电。如此推其完整视频播放续航能力大概也是在9小时左右。顺便，当时是先使用Surface Book 2的平板部分一口气连看三集《Unnatural》，然后平板才提示充电，再插回底座继续看。
　　轻薄本颜值游戏本的心 无可争议王者再临
　　平心而论，评判Surface Book 2为其定性是一件非常非常困难的事情，因为它自身就是参考的标准——上一代Surface Book就是如此，它设立了Windows阵营里性能型笔记本的模范；而这一回Surface Book 2的登场重新设立全能笔记本的标准，而且还顺便教育了一下隔壁那些游戏本什么叫真正的笔记本美学。
　　就是如此，Surface Book 2继承家族特性高颜值，在优雅而冷俊的外表下还藏着丝毫不输主流游戏笔记本的强大性能，它就像贵族家中那位华丽得体的大管家，可以胜任一切你交给它完成的任务。聘请它为你服务可能会花费不少金钱（顶配版本26488元），但这一切都物有所值，Surface Book 2所能给你带来的优秀体验均衡而又到位。如果你正在寻找一部最接近完美的笔记本，Surface Book 2就是那位唯一的梦中情人。
